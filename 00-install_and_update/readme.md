# Install

使用 `rustup` 工具，安装非常方便

在 windows 上，我是用 customlize install 选项将工具链切换为 `x86_64-pc-windows-gnu` ，这是更加极客的选项，我很喜欢

# Update

使用 `rustup `工具

```shell
rustup update
```

# Uninstall

使用 `rustup `工具

```shell
rustup self uninstall
```

# local document

使用 rustup 工具可以打开一份本地文档，非常好用

```shell
rustup doc
```

总结下来，`rustup `非常方便，非常好用
